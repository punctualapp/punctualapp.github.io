# punctualapp.github.io

Landing page for **Punctual** — a free, open-source macOS menu bar app that makes sure you're never late to a meeting.

Live at [punctualapp.github.io](https://punctualapp.github.io).

## How it works

This is a plain static site (a single `index.html`, no build step). GitHub Pages serves the root of the `main` branch automatically — push to `main` and the site updates within a minute or two.
